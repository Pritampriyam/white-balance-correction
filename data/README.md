# Data Directory

This directory contains the dataset for white balance correction.

## Structure

- `raw/` - Original, unprocessed data
  - `train/` - Training data
    - `images/` - 2,539 TIFF images (256x256)
    - `sliders.csv` - Training targets and metadata
  - `validation/` - Validation data  
    - `images/` - 493 TIFF images (256x256)
    - `sliders_inputs.csv` - Validation metadata

- `processed/` - Processed data (generated by scripts)

## Data Format

### sliders.csv
- `id_global` - Unique image identifier
- `currTemp` - Current temperature value
- `currTint` - Current tint value  
- `Temperature` - Target temperature (training only)
- `Tint` - Target tint (training only)
- Other EXIF metadata columns

### sliders_inputs.csv
- Same as sliders.csv but without Temperature/Tint targets

## Preprocessing

Run `python scripts/setup_data.py` to prepare the data for training.
